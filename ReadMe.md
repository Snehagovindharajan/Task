Scientific Calculator
The Scientific Calculator application is written in Python language. The app provides standard and scientific
functionalities.

Features
    Standard Calculator functionality which offers basic operations and evaluates commands immediately as they are entered.
    Scientific Calculator functionality which offers expanded operations and evaluates commands using order of operations.

Getting Started


I have worked on the function x power y and written the logic to find power of any positive or negative numbers.
I have writtern test cases to check when
    1) base and power are positive
    2) base is negative and power is positive
    3) base and power is negative
    4) base is positive and power is negative
    5) base is string and power is a number and vice versa
